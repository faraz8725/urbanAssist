Project Description

Problem:
Finding reliable local service providers like plumbers, electricians, tutors, or delivery agents is often difficult and time-consuming. People rely on word-of-mouth or unverified sources, which leads to trust issues, delays, and inconsistent service quality.

Solution:
UrbanAssist is a full-stack web platform that connects customers with verified local service providers in their neighbourhood. It provides a seamless way to discover, book, and review services in real-time, ensuring trust, convenience, and efficiency.

Key Features:

- Search & discover nearby service providers
- Easy booking system with time slots
- Ratings & reviews for trust building
- Service provider profiles with skills & pricing
- Real-time chat between customer & provider
- Secure authentication & user management

Tech Stack:
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT
Deployment: Vercel / Render

Impact:
UrbanAssist helps users find trusted services quickly, improves service reliability, and creates earning opportunities for local providers.
